# Sales Dashboard 📊

An interactive sales dashboard built with Python and Streamlit.

## Live Demo
https://nandhini--sales-dashboard.streamlit.app/

## What it does
- Analyzes 4 years of Superstore sales data (2015-2018)
- Interactive filters by Year, Category and Region
- KPI metrics — Total Sales, Total Orders, Avg Order Value
- Sales forecast for next 6 months using Facebook Prophet

## Tech Stack
- Python, Pandas
- Plotly (interactive charts)
- Streamlit (dashboard)
- Facebook Prophet (forecasting)

## Key Insights
- Technology is the top category at $827k total sales
- West region leads with 31.4% of total sales
- Phones and Chairs are the best selling sub-categories
- Sales show consistent year-end spike every November/December

## How to run
pip install -r requirements.txt
streamlit run app.py
